# job-tracker-app-api
Job tracker API project.
> This application is intended to track jobs to provide ease of mind when applying for jobs.

## Overview
Full stack application built with the Vue framework for the frontend and the Django framework for the backend.
The goal of this project is to build a job search application that fetches job listings that will intergrate with 
external job search APIs (e.g., Indeed, Adzuna).

This will allow users to browse, filter, and save job listings without visiting several job search sites.

## Tech Stack
**Frontend:** Vue
**Backend:** Django
**Database:** MySQL
**Other:** Docker
